# 🚀 AWS Static Website Hosting Project

## 📌 Project Overview
This project demonstrates how to host a static website using AWS S3 and deliver it globally using CloudFront.

## 🛠️ Services Used
- AWS S3
- AWS CloudFront
- IAM
- GitHub

## ⚙️ Steps
1. Created S3 bucket
2. Uploaded website files
3. Enabled static hosting
4. Configured bucket policy
5. Setup CloudFront CDN
